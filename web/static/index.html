⚙️ Setup — Dành cho người mới

🌐 Portal tuyển dụng IT VN

Click vào link để mở trang tìm việc, đăng ký tài khoản nếu cần.

✅ ITviec — itviec.com LinkedIn LinkedIn VN 🌐 Vieclam24h Free Freehire (cần API key miễn phí)

👤 Facebook Clone Account

Cần acc clone để crawl job từ group FB (tránh rủi ro khóa acc chính).

  1. Tạo email mới (không liên kết với acc FB chính)
  2. Đăng ký FB với tên tiếng Việt, thêm ảnh đại diện
  3. Dùng acc tự nhiên 3-5 ngày: like, comment, kết bạn
  4. Join các group IT: AI/ML Vietnam, Computer Vision Vietnam, Data Science VN
📖 Xem hướng dẫn chi tiết

Xem file web/warmup_guide.md trong project để biết chi tiết từng bước warm-up.

Sau khi warm-up xong, chạy lệnh:

crwl profiles

Chọn "Create new profile" → đặt tên fb-clone → login acc clone → nhấn q để lưu.

🛠️ Cài đặt công cụ

Chạy các lệnh sau trong terminal:

1. Cài crawl4ai (Facebook crawler)
pip install crawl4ai && crawl4ai-setup
2. Cài Bun (JavaScript runtime)
curl -fsSL https://bun.sh/install | bash
3. Cài deps cho ITviec CLI
cd .agents/skills/itviec-search/cli && bun install
4. Khởi động dashboard
cd web && python3 server.py

💡 Mẹo sử dụng

  • 🔍 DB = tìm trong database đã crawl trước đó (nhanh)
  • 🌐 Live Search = crawl trực tiếp từ portal (chậm hơn, data mới nhất)
  • Click vào job card để lưu nhanh vào danh sách ứng tuyển
  • Chọn portal Facebook rồi Live Search để crawl group FB
  • Dashboard tự động dedup job trùng (cùng source_id + portal)
  • Theme sáng/tối: toggle ở góc phải trên cùng ☀️🌙
